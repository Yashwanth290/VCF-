VCF Processing and Analysis Toolkit

This repository contains scripts and workflows for processing, analyzing, and interpreting Variant Call Format (VCF) files.
It includes tools for filtering variants, extracting genomic regions, annotation, and generating summary statistics for downstream analysis.
